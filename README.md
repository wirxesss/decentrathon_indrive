Проект: Система скоринга для inDrive AI (Decentrathon 5.0)

Выбранный кейс:
Кейс 2: Скоринг сельхозпроизводителей при получении субсидий.

Описание решения:
Система анализирует данные фермеров и выставляет баллы на основе их эффективности. 

Главная цель: сделать распределение субсидий справедливым и прозрачным.

Архитектура:
1. Ввод данных (Excel/CSV с показателями фермеров).
2. Обработка (Python-скрипт с логикой начисления баллов).
3. Результат: Рейтинг кандидатов с объяснением оценки (Explainability).
